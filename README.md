# Whack-a-mole
Uppgift till FWMX25, där vi skulle skapa ett Whack-a-mole spel.
## Funktioner
- Dynamiskt 3×3 spelbräde skapat i JavaScript  
- Flexbox-baserad layout  
- Mullvadar som dyker upp i slumpmässiga hål  
- TTL (time-to-live) — mullvaden försvinner efter en viss tid  
- Klicka mullvaden för +1 poäng  
- Klick bredvid ger miss  
- Tangentbordsstöd: Tab + Enter/Space  
- Eventdelegering från spelbrädet  
- Start, Pause/Resume och End Game  
- Timer-nedräkning (60 sek)
- Modulär ES6-struktur
## Instruktioner
Klona repot
Öppna index.html med live-server
Klicka på "Starta spel"
Försök att träffa alla mullvader som dyker upp, du har 60 sek på dig.
